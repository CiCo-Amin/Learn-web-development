Bab 1 - Elemen Teks

HTML digunakan untuk menyusun struktur sebuah halaman web. Dengan HTML kita dapat menampilkan berbagai macam jenis teks.

"Belajar coding bukan tentang menghafal syntax, tetapi memahami konsepnya."

Contoh kode HTML:

<h1>Halo Dunia</h1>

Ini adalah teks tebal.

Ini adalah teks miring.

Ini adalah teks bergaris bawah.

Ini adalah teks dengan ukuran lebih kecil.

Rumus luas: m2

Rumus air: H2O

Pelajari bagian yang paling penting terlebih dahulu.

HTML adalah bahasa markup untuk membuat website.
